Kafka Components (Simplified)
1. Broker - The Post Office
Kafka server that stores and manages messages
2. Topic - Mailbox Category
Categories for messages (e.g., "orders", "notifications")
Messages are published to topics
3. Producer - Sender
Services that send messages to topics
4. Consumer - Receiver
Services that read messages from topics
5. Zookeeper - Manager
Manages and coordinates Kafka brokers
(Note: Newer versions are removing Zookeeper)
Kafka objects that we have to create in order for us to deal with kafka form sender perspective

-Topics (Message mailboxes)
  Topics are created automatically when you first send a message.
  @Bean
  public NewTopic orderTopic() {
      return TopicBuilder.name("orders")
              .partitions(1) //how many subfolders
              .replicas(1) //how many back up copies
              .build();}
